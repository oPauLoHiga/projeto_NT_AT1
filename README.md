# Projeto Novas Tecnologias

Projeto da disciplina Novas Tecnologias, referente a uma atividade avaliativa em grupo.

Com os dados obtidos por meio de um formulário, transformados em arquivo .csv e devidamente tratados, poderemos realizar uma análise sobre os hábitos de estudo, sono e produtividade de universitários.

Para isso, será utilizado Python, sem o uso de bibliotecas que auxiliem no tratamento ou na análise dos dados.
